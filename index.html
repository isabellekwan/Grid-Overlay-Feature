Links can be recategorized by clicking the category label buttons located at the bottom of each link. Ensure you refresh your page to see the changes.

Additional Information

The number of links under each category and the amount of time spent at all the links can be found at the top right corner. As well, the tag attached to each link can be found next to the "helpful" button.

What are Events?

Events are records of resources you have visited that have been tagged or bookmarked. Events automatically appear on your calander with their website title and tag. Their default duration is set as 1 hour and this will grow as more time is spent.

Editing Events

Event titles can edited by clicking once on the existing title and typing. The duration can be changed by dragging the event's bottom edge up or down.

Creating a New Event

You can create new events by clicking and dragging down on an empty spot on the calander.

Moving Events

Events can be moved by clicking and holding on an event and dragging it to a different location.

Changing Event Display

Double clicking an event will change the event view to the next time display. For example, double clicking a month will change the event view to display in weeks.

Label Colours

The colour of each event indicates which category the website falls under. These can be changed in the Sortify overview.
